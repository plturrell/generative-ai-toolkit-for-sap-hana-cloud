Accelerated Features

Advanced NVIDIA GPU optimizations to supercharge your SAP HANA AI workloads

TensorRT Optimization

Advanced deep learning inference optimizer with kernel fusion, precision calibration, and dynamic shape support for up to 3x faster inference.

  • FP16/INT8 precision support
  • Kernel fusion and optimization
  • Automatic engine optimization

NGC Container Publishing

Pre-optimized container images published to NVIDIA NGC for simplified deployment and enterprise GPU environment integration.

  • Enterprise-ready containers
  • Automated CI/CD publishing
  • GPU-optimized dependencies

Hopper Architecture Support

Advanced optimizations for NVIDIA H100 GPUs including FP8 precision, Transformer Engine, and Flash Attention 2 integration.

  • FP8 precision support
  • Transformer Engine integration
  • Flash Attention 2 acceleration
